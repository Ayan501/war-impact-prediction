# War Impact Prediction 

A Machine Learning project that predicts the economic 
and human impact of wars and conflicts on common people.

##  Problem Statement
Predicting how wars affect GDP, inflation, commodity 
prices and common people's lives using historical data.

## Dataset Sources
- UCDP Conflict Data (1946-2024)
- World Bank Inflation Data
- Commodity Prices (Crude Oil, Gold, Wheat, Sugar)

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn, XGBoost
- Flask, React.js
- SQLite
